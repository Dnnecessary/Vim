### Install
1. clone the repo
```
https://github.com/Dnnecessary/my_vim.git
```
</br>

2. clone the repo
```
mv my-vim ~/.vim
```
</br>

3. set up `~/.vimrc`, have a fake `.vimrc` in your `$HOME`
```
 cd  ~/.vim
 cp  vimrc.bak   ~/.vimrc
```
