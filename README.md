### Install
1. clone the repo
```
git clone https://github.com/Dnnecessary/Vim.git
```
</br>

2. clone the repo
```
mv my-vim ~/.vim
```
</br>

3. set up `~/.vimrc`, have a fake `.vimrc` in your `$HOME`
```
 cd  Vim
 cp  vimrc   ~/.vimrc
```
